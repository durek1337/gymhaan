# Schulbetriebssystem
## Installation
### Betriebssystem
* Ubuntu Version 18/19 ?
### Scripts
Die Scripts im Ordner "scripts/" müssen ggf. noch ausführbar gemacht werden.
```
cd scripts
chmod +x *.sh
```
Danach mit root-Rechten starten und zwischendurch auf notwendige Interaktionen reagieren
```sudo ./startscript.sh```
